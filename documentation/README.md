# Rufus Production Documentation

This directory contains operational documentation for Rufus, the production entity for koad:io.

## Structure

### Operational Guides
- `OPERATIONS.md` — Production workflows, session setup, and asset pipeline
- `HARDWARE_MANIFEST.md` — Equipment inventory, capabilities, and network topology
- `OBS_SETUP.md` — OBS scene templates, configuration, and multi-source routing

### Technical Specifications
- `encoding-spec.md` — Codec, bitrate, and quality standards for all content types
- `naming-conventions.md` — Asset naming and organization schema
- `export-formats.md` — Platform-specific export standards (YouTube, podcasts, archive)

### Production Playbooks
- `workflows/` — Type-specific playbooks (interview, podcast, live-stream, screen-recording)
- `checklists/` — Pre-recording, during-recording, and post-production checklists
- `troubleshooting/` — Common issues and recovery procedures

### Team Guides
- `for-faber.md` — Guidance for content creators (Faber)
- `for-mercury.md` — Asset delivery and publication guidelines (Mercury)
- `for-operators.md` — Hardware operation and monitoring (thinker, flowbie, fourty4)
